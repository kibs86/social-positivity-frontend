MY SOCIAL SENTIMENT

Bringing social media sentiment analysis to a personal level.

info_outline

General Informationmore_vert
General Informationclose

This app analyzes text from your Facebook timeline or from text you input and shows you the positivity/negativity of keywords found within that text.

help_outline

How It Worksmore_vert
How It Worksclose

The Facebook API is used for authentication and to pull in the timeline for the user. The IBM Watson Alchemy Language API is used to analyze the sentiment of target phrases found within the text provided.

star_outline

Future Enhancementsmore_vert
Future Enhancementsclose

Future versions of this application will include integration with Twitter and other social media applications.
